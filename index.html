Our Concept

Hollow Life

About The Scene

In our scene the yellow sphere represents the sun, while the purple sphere represnts the life at Habib University. The scene depicts the idea that for a student their time at habib is the most enlightening period they will ever experience. Due to this only the purple sphere is shining but the rest of the part is dull.

Image Features

Blinn-Phong

Blinn-Phong shading used to create materials that have speular reflection.

BRDF

BRDF used in Blinn-Phong shading, and the function itself can be modified to control the shininess of the material.

Ray Casting

Ray Casting was used to simulate shadows based on the direction of the incident light on the object.

Code Features

Lighting

In our project we have implemented four types of light: Ambient Light, Directional Light, Point Light, and Spot Light. A combination of lights can be used to give a new meaning to the same old scene

Acceleration

We have implemented and are using Bounding Volume Hierarchy as the acceleration structure to speedup our rendering process.

Camera

The project contains two types of cameras: Parallel and Perspective. Either of the camera could be used to get a different view.

Geometry

Our Raytracer supports plane, sphere, and triangle as the basic geometry primitives.

Mesh Importer

Our code is compatible to load any ply mesh which does not have a color value already assigned. The loaded mesh can be rendered for different resolutions.

Material

In the project we have implemented Blinn Phong Model, which can be used to create different types of material just by varying diffuse and specular components.

Acceleration structures

In this raytracer we have implemented Bounding Volume Hierarchy as the acceleration structure to speedup our rendering process. The table below shows time taken to render our scene both low and high quality, with and without acceleration structure. We can clearly see that the process has become faster.

With Acceleration structures Resolution Render Time (in seconds)
False 500x500 (low) 1.487
True 500x500 (low) 1.487
False 2000x2000 (high) 22.625
True 2000x2000 (high) 22.625
